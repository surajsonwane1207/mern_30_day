# Day 1: HTML Basics

## Overview
Learn the fundamentals of HTML and build a simple webpage.

## Topics Covered
- HTML structure and syntax
- Common HTML tags
- Semantic markup
- Forms and inputs
- File structure and organization
- Best practices

## Learning Objectives
- Understand HTML document structure
- Create a valid HTML webpage
- Use semantic HTML elements
- Build a simple form

## Project: Simple Webpage
Build a personal profile webpage with:
- Header with navigation
- About section
- Contact form
- Footer

## Key Concepts
- `<!DOCTYPE>`
- `<html>`, `<head>`, `<body>`
- Heading tags (`<h1>` - `<h6>`)
- Paragraph and text elements
- Links and images
- Lists (ordered and unordered)
- Semantic elements (`<header>`, `<nav>`, `<section>`, `<footer>`)

## Resources
- [MDN HTML Documentation](https://developer.mozilla.org/en-US/docs/Web/HTML)
- [W3C HTML Standard](https://html.spec.whatwg.org/)

## Challenge
Create an HTML page with proper structure that displays information about yourself.
